#Проект по автоматизации API тестов для  демо-приложения EZ-scooter( аренда самокатов)
## Документация: https://qa-scooter.praktikum-services.ru/docs/
### Использованные технологии:
+ Java
+ Maven
+ Rest Assured
+ Junit 5
+ Allure

Запуск тестов:
> maven clean test

Allure отчет:
> mvn allure:serve
